# ColorfulNotesApp

Android App with Firebase Integration
Features:
-> User Registration and Login.
-> Create, Edit, and Delete Notes.
-> Colorful Notes.

Technologies Used:
--> Android Studio
--> Java
--> Firebase Authentication (for user registration and login)
--> Firebase Firestore (for storing notes)
